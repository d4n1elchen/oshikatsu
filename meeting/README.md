# Meeting Notes

This folder contains meeting records for the Oshikatsu project.

## Purpose

Use this folder to record discussions, decisions, artifacts, and action items for each individual meeting.

## Structure

Each meeting should have its own subfolder under `meeting/`, for example:

- `meeting/2026-04-15-initial/`
  - `notes.md`
  - `artifacts/`

## Recommended Meeting Folder Contents

- `notes.md`: Main meeting notes file for that session.
- `artifacts/`: Optional folder for related files, diagrams, screenshots, or supporting materials.

## Meeting Notes Format

Each meeting `notes.md` should include:

- `Topics`
- `Decisions`
- `Artifacts`
- `Action Items`

This layout ensures every meeting has its own dedicated record and keeps follow-up items easy to track.
